# Gestao-Nutri
Aplicação web para gestão de uma clínica de nutrição

---

# Descrição do Projeto
A aplicação será desenvolvida em JavaEE com o objetivo de facilitar a gestão diária de uma clínica de nutrição. A plataforma permitirá aos profissionais de nutrição armazenar, consultar, atualizar e analisar de forma eficiente toda a informação relativa aos seus clientes e respetivas consultas. O sistema contemplará funcionalidades como registro de dados antropométricos, criação/atualização de planos alimentares bem como acompanhamento da evolução do cliente ao longo do tempo.
